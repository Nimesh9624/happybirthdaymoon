# 🎂 Happy Birthday Moon

A special birthday surprise website created for Moon.

## Features

- 📅 Countdown Calendar (June 1 - June 15)
- ✨ Daily page reveal animation
- 💌 Special Birthday Message
- 🎉 Popup Surprise Message
- 🎂 Interactive Cake Cutting
- 🕯️ Birthday Candles
- 🎵 Background Music
- 📸 Photo Gallery
- 🎊 Confetti Celebration Effects
- 📱 Mobile Responsive Design

## Technologies Used

- HTML5
- CSS3
- JavaScript
